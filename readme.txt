== Description ==

Adds Call, Email and SMS buttons on bottom of website. Only for Mobile View of website.
The plugin will use default call, text and email providers installed on user phone.

== Installation ==

1. Upload the plugin file mobile-contact-buttons.zip to the `/wp-content/plugins/` directory and extract it, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Appearance->Customize->Mobile Contact Buttons screen to configure the plugin


== Frequently Asked Questions ==

= How to change background color of buttons? =

Go to Appearance->Customize->Mobile Contact Buttons and under 'Background Color' set the color.

= How to change phone number for call? =

Go to Appearance->Customize->Mobile Contact Buttons and under description 'Phone Number for Calls' set the phone nummber. You must include country code as well.

= How to change phone number for sms? =

Go to Appearance->Customize->Mobile Contact Buttons and under description 'Phone Number for SMS' set the phone nummber. You must include country code as well.



= Can I remove SMS(Text) button and leave other two buttons? =

Yes.It is includded in this version.
